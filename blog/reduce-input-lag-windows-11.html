How to Reduce Input Lag in Windows 11: 12 Proven Tweaks

Windows 11 ships with background services, visual effects, and power management settings that silently add milliseconds of input lag to every keypress and mouse movement. Here's how to strip them out systematically.

What Causes Input Lag in Windows 11

Input lag is the total time between a physical input (mouse click, key press) and the visible result on screen. It's composed of several layers: hardware polling latency, OS processing delay, GPU render time, and monitor display time. Windows 11 adds to the OS layer through background tasks, visual compositing (DWM), and power throttling.

The good news: most of these are configurable. A fully optimized Windows 11 install can reduce the software portion of input lag by 5–15ms compared to out-of-box defaults — a meaningful improvement at 144Hz+.

Switching to Ultimate Performance power plan prevents CPU clock throttling that adds inconsistent processing delay.

Tweak 1–2: Power Plan Settings

  1. Enable Ultimate Performance power plan: Open PowerShell as admin and run: powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61. Then set it active in Control Panel → Power Options.
  2. Disable CPU Core Parking: In the active power plan, set "Processor performance core parking min cores" to 100% — this prevents CPUs from parking cores that need to wake up to process input events.

Tweaks 3–8: Disable Windows Features That Add Latency

  1. Disable Xbox Game Bar: Settings → Gaming → Xbox Game Bar → Off. Game Bar hooks into game processes and adds overhead.
  2. Disable Game Mode: Counterintuitively, Windows Game Mode can cause stutters on some systems. Settings → Gaming → Game Mode → Off. Test with and without.
  3. Disable Hardware-Accelerated GPU Scheduling (HAGS): For older GPUs (< RTX 3000 series), HAGS adds latency. Settings → System → Display → Graphics → Change default graphics settings.
  4. Disable Visual Effects: Search "Adjust the appearance and performance of Windows" → select "Adjust for best performance."
  5. Disable Nagle's Algorithm: This TCP optimization buffers network packets and adds latency in online games. Registry tweak required — search "disable Nagle's algorithm Windows 11" for the specific registry path.
  6. Set mouse polling rate to maximum: In your mouse software, set polling rate to 1000Hz (or 4000Hz/8000Hz if your mouse supports it). Higher polling = more frequent position updates.

NVIDIA Control Panel: set Low Latency Mode to Ultra and Max Pre-Rendered Frames to 1 for minimum GPU-side buffering.

Tweaks 9–12: GPU Driver Settings

  1. NVIDIA: Low Latency Mode = Ultra (NVIDIA Control Panel → Manage 3D Settings). Queues frames right before the GPU needs them, minimizing pre-render buffer.
  2. NVIDIA: Max Pre-Rendered Frames = 1. Reduces the GPU frame buffer that delays display of completed frames.
  3. Disable VSync in GPU driver panel (unless using G-Sync). Traditional VSync doubles or triples input lag.
  4. Enable NVIDIA Reflex or AMD Anti-Lag in supported games. These engine-level latency reducers are the most effective single tweak available — they can reduce total system latency by 30–50% in supported titles (CS2, Valorant, Apex, Overwatch 2).
