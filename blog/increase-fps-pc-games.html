How to Increase FPS in Modern PC Games: 15 Tweaks That Actually Deliver Results

Knowing how to increase FPS in PC games is the difference between playing at your hardware's actual ceiling and playing 30–50% below it. Most gaming PCs are not properly optimized — they're hamstrung by default Windows settings, conservative GPU driver configurations, and in-game visual options that consume enormous resources while contributing almost nothing to competitive performance. These 15 tweaks address all of that.

Step 1: Identify What's Actually Limiting Your FPS

Before changing any settings, open a monitoring overlay (MSI Afterburner with RivaTuner, or the in-game overlay in GeForce Experience) and check your GPU usage and CPU usage simultaneously during gameplay. You need to know which component is the bottleneck:

  • GPU at 98–100%, CPU below 70%: You're GPU-limited. Reducing graphical settings will increase FPS.
  • CPU at 90–100%, GPU below 80%: You're CPU-limited. Reducing game resolution or CPU-heavy settings (draw distance, player count) will help more than texture quality changes.
  • Both below 70%, but low FPS: You have a bottleneck elsewhere — often RAM speed, a dying drive, or a driver issue.

This diagnostic step saves hours of ineffective tweaking. Changes that help a GPU-limited system do nothing for a CPU-limited one.

In-Game Settings That Increase FPS the Most

Tweak 1: Reduce Resolution Scale / Render Scale

Render scale is the single most GPU-expensive setting. Dropping from 100% to 75% render scale reduces GPU load by approximately 44% (since you're rendering 44% fewer pixels), which translates to massive FPS gains. In Valorant this is under Video → Resolution. In CS2 it's set via the launch option or resolution menu. This is the nuclear option for FPS gains.

Tweak 2: Disable Anti-Aliasing or Switch to FXAA/TAA

MSAA (Multi-Sample Anti-Aliasing) at 4x or 8x is one of the most GPU-intensive settings in any game. Disable it, or switch to FXAA (a post-process technique with minimal performance cost). In competitive games, clean edges matter less than a high, stable FPS anyway.

Tweak 3: Set Shadows to Low or Very Low

Dynamic shadow rendering is extremely expensive. Setting shadows to Low or Very Low in most modern games can recover 15–30% FPS with very little visual impact in competitive scenarios. Dynamic shadows don't help you spot enemies — enemy models do. Lower shadows, more frames.

Tweak 4: Reduce Draw Distance / Level of Detail to Medium

Object draw distance and LOD (Level of Detail) settings determine how many distant objects are rendered at full detail. In open-world games like Warzone or Apex, reducing this significantly cuts CPU draw call processing. Set to Medium as a starting point.

Tweak 5: Disable Ambient Occlusion

Ambient occlusion (SSAO, HBAO+) calculates complex lighting interaction at surface intersections. It looks great, costs 10–20% FPS, and makes zero difference to your competitive awareness of the game environment. Disable it.

Tweak 6: Set Texture Quality to Medium (Not High)

High texture quality primarily stresses VRAM, not GPU processing power. If you have 8GB+ VRAM, this may not cost any FPS. But if you're on a 6GB card playing a modern title, setting textures to Medium frees VRAM for frame buffers and can stabilize 1% low FPS significantly.

GPU Driver Settings That Directly Increase FPS

Tweak 7: Update Your GPU Drivers

GPU manufacturers release driver updates specifically optimized for new game releases. Running a driver from 6 months ago on a recently launched game means you're not getting the performance optimizations Nvidia or AMD wrote specifically for that game. Update via GeForce Experience (NVIDIA) or AMD Software: Adrenalin Edition.

Tweak 8: Enable Resizable BAR (NVIDIA) / Smart Access Memory (AMD)

Resizable BAR allows the CPU direct access to the GPU's full VRAM rather than 256MB chunks. This eliminates a memory transfer bottleneck and can improve performance by 5–15% in supported games. Enable in BIOS (look for "Above 4G Decoding" + "Resizable BAR Support") and verify in GPU driver settings.

Tweak 9: Overclock Your GPU (Safe, Software-Based)

Using MSI Afterburner, increase your GPU core clock by +50–100 MHz and memory clock by +200–500 MHz incrementally, running a stability test (30 minutes of gaming) between increments. A stable +5–15% GPU overclock is achievable on most modern GPUs and translates directly into FPS gains.

Windows System Tweaks

Tweak 10: Enable Windows Game Mode

Settings → Gaming → Game Mode → On. Game Mode prioritizes GPU and CPU resources for the active game process. In Windows 11, it's improved substantially over earlier versions and genuinely helps consistency.

Tweak 11: Disable Full-Screen Optimizations for Your Game Executable

Right-click your game's .exe file → Properties → Compatibility → Disable fullscreen optimizations. This allows the game to take exclusive control of the display pipeline, reducing compositor overhead. Combined with running your game in exclusive fullscreen mode (not borderless windowed), this gives the lowest possible display latency and often a small FPS boost.

Tweak 12: Upgrade to an NVMe SSD if Still on HDD

A spinning hard disk doesn't limit your FPS directly, but it causes dramatic frame time spikes when the game streams assets from disk during gameplay. Texture streaming stutters, level transition lags, and 1% lows all improve dramatically on NVMe storage. This is a hardware upgrade recommendation, but it's by far the highest-impact storage investment.

Upscaling Technologies: Free FPS With Minimal Quality Loss

Tweak 13: Enable DLSS (NVIDIA RTX GPUs)

DLSS (Deep Learning Super Sampling) uses dedicated Tensor Cores on RTX GPUs to reconstruct a higher-resolution image from a lower-resolution render. At DLSS Quality mode, most players cannot visually distinguish the output from native resolution — yet FPS gains of 40–70% are typical. If your game supports DLSS, enable it.

Tweak 14: Enable FSR 2 / FSR 3 (All GPUs, AMD and NVIDIA)

AMD's FidelityFX Super Resolution 2/3 is an open standard that works on any GPU. FSR Quality mode gives visually clean output with 25–50% FPS improvements in supported titles. Available in many games' video settings menus.

When Software Can't Do More: Hardware Upgrades

Tweak 15: Upgrade Your RAM Speed and Enable XMP/EXPO

Many gaming PCs run DDR4 or DDR5 RAM at stock speeds well below their rated maximum — because XMP/EXPO profiles aren't enabled in BIOS. Enter BIOS, find "XMP" (Intel) or "EXPO" (AMD), and enable the highest stable profile. Going from 2133MHz to 3200MHz DDR4 can improve CPU-limited game performance by 10–20% because modern CPUs (especially AMD Ryzen) are highly sensitive to RAM latency and bandwidth.
